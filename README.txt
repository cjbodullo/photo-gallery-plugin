A simple gallery plugin with shortcode support.

== Installation ==

1. Copy `photo-gallery-plugin` folder into `wp-content/plugins/`
2. Activate plugin from WordPress admin.
3. Use shortcode: `[winner_photo_gallery limit="12" columns="4" order="DESC"]`

== Shortcode ==

- `ids`: comma-separated WordPress media attachment IDs
- `columns`: 1 to 6 (default: 3)
